# AWS Deployment - Batch Architecture

## Project summary

This deployment aims to implement a big data storage and processing ecosystem using data from the integrator project `who_life_exp.csv`, developing a lifecycle based on the basic batch architecture in AWS.

## Files in this folder

- **`Notebooks/`**:
  * `Data prep.ipynb`
  * `EDA.ipynb`
  * `Train Model.ipynb`
  * `Visualizaciones.ipynb`
-  **`Reproducibility_Guide.pdf`**: Guide for reproducing the project through the AWS web interface.

## Notes & reproducibility

To reproduce the results:

1. Ensure you have access to the AWS web interface and your AWS credentials.
2. Follow the reproducibility guide located in the "Documentation" folder.
